# 🏋️‍♂️ GymGuide

Plataforma web voltada para auxiliar usuários na organização de treinos, alimentação e evolução física.

## 🚀 Sobre o projeto

O GymGuide foi desenvolvido como projeto de portfólio com foco em desenvolvimento fullstack, oferecendo ferramentas práticas para quem busca melhorar desempenho físico e consistência nos treinos.

A aplicação reúne:
- 📊 Calculadora de calorias e macronutrientes
- 🍽️ Simulador de dieta com alimentos reais
- 🏋️ Lista de exercícios com filtros por grupo muscular
- 📅 Gerador de fichas de treino
- ⭐ Sistema de favoritos (em desenvolvimento)
- 🔐 Sistema de autenticação (em desenvolvimento)

---

## 🧠 Funcionalidades

### 🔹 Treinos
- Filtro por grupo muscular
- Filtro por nível (iniciante, intermediário, avançado)
- Sugestão automática de exercícios

### 🔹 Dieta
- Cálculo de calorias baseado em TMB
- Distribuição de macronutrientes
- Simulação de refeições com cálculo automático
- Comparação com meta diária

### 🔹 UX/UI
- Interface moderna e responsiva
- Animações suaves
- Seções colapsáveis para melhor organização

---

## 🛠️ Tecnologias utilizadas

### Frontend

- JavaScript (Vanilla)

### Backend (em construção)
